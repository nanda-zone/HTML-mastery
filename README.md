# Single-page CV - Semantic HTML Project

This project is an implementation of single-page curriculum vitae with a focus on **Semantic HTML** structure and **SEO optimization**. It serves as a solid foundation for future styling and web development projects.

## 🚀 Live Demo
You can view the live version of my CV here:  
[https://nanda-zone.github.io/HTML-mastery/single-page-cv.html]

## 🛠️ Project Requirements & Features
This project fulfills the following technical specifications:
- **Semantic HTML**: Using meaningful tags (like `<header>`, `<main>`, `<section>`) for better accessibility and structure.
- **SEO Meta Tags**: Essential metadata included to improve search engine discoverability.
- **Open Graph (OG) Tags**: Optimized for professional social media sharing on platforms like LinkedIn and WhatsApp.
- **Favicon**: A custom browser icon to enhance branding.
- **Ready for Styling**: A clean and organized class-based structure, prepared for future CSS implementation.
